# Java Studies

## Habilidades técnicas estudadas?
- Framework Spring Boot.
- Gerenciador de dependências - Maven e Gradle.
- Testes com JUnit5.
- Banco de dados SQL.
- ORM utilizando Hibernate Framework.
- Estrutura Coleções.
- Deploy utilizando Railway.
- Docker.

## Estrutura de Diretórios e Projetos Estudados

Organizei este repositório em módulos principais, cada um contendo os exercícios e projetos que desenvolvi durante a minha jornada de aprendizado na aceleração:

### 1. Fundamentos Java (`/Fundamentos-java`)
Neste módulo, eu aprendi os princípios básicos da linguagem Java, ferramentas de build e lógica de programação. Eu adquiri a habilidade de:
- **Debugging:** Depurar, analisar código e corrigir erros na prática.
- **Métodos de String:** Explorar e manipular textos usando os métodos da classe `String`.
- **Exercício Conta Bancária (Gradle e Maven):** Criar uma aplicação de conta bancária, o que me deu a habilidade de introduzir e configurar os gerenciadores de dependência `Gradle` e `Maven`.
- **Exercício Escola Inteligente:** Desenvolver a lógica para um sistema escolar, consolidando meu conhecimento em estruturas de controle e fluxo.
- **Projeto Regra de Progressão:** Implementar uma calculadora de notas e progresso escolar, avaliando o desempenho com base em pesos e médias.

### 2. Programação Orientada a Objetos (`/POO`)
Foquei na construção de sistemas baseados nos pilares da Orientação a Objetos (Encapsulamento, Herança, Polimorfismo e Abstração). Eu aprendi a:
- **Exercício Gestão de Veículos:** Modelar um sistema de frota ou locadora, aplicando na prática conceitos de herança e interfaces para diferentes tipos de veículos.
- **Exercício Lista POO:** Criar classes, instanciar objetos e definir atributos e métodos através de uma bateria de exercícios práticos.
- **Projeto Sistema de Votação:** Construir uma aplicação completa para simular uma eleição, gerenciando pessoas eleitoras, candidatas e a contabilização de votos de forma puramente orientada a objetos.

### 3. Java Web (`/JavaWeb`)
Construí aplicações e APIs modernas utilizando o ecossistema Spring Boot, banco de dados e testes. Eu adquiri a habilidade de:
- **Calculadora & Calculadora de Idade:** Criar rotas web e manipular requisições e respostas básicas em APIs.
- **Exercício Caixa Eletrônico:** Desenvolver uma simulação web de operações financeiras, lidando com lógica de transações, saques e depósitos em endpoints.
- **Exercício Gerenciador de Tarefas:** Criar um CRUD (Create, Read, Update, Delete) completo para o gerenciamento de um To-Do list via API REST.
- **Exercício Relatório de Clima:** Estruturar APIs voltadas para o consumo e disponibilização de dados climáticos e meteorológicos.
- **Exercício Fitness API:** Desenvolver uma API RESTful para o registro e gerenciamento de perfis e rotinas de atividades físicas.
- **Projeto Localizador de Museus:** Estruturar uma API mais complexa com regras de negócios, focada no cadastro de museus e de suas informações (localização, temas, etc.).
